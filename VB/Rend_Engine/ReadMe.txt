This sample shows how to create a Rendition Engine using IDM Desktop SDK.
NOTE: This project REQUIRES Office 2000, it WILL NOT compile on Office 97.
It uses Office 2000 to convert documents to HTML format.

The following areas of IDM Desktop SDK are covered by this sample:

* Creating and registering a Rendition Engine.
* Creating Style Templates.
* Querying library for publish requests using a timer control.
* Processing the publish requests.
* Adding contents to the published documents.
* Creating compound documents.
* Versioning compound documents.
* Marking the document as "published" when done processing.

Please review the code for more information.
The comments will guide you through the process of creating a custom Rendition Engine.
